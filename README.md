# ProjectFinal-INST377
Project final deliverable for INST377. This is a job search web app designed for women of color. 

**Project Title:**
EmpowerLink

**Project Description:**
EmpowerLink is a web application designed to help women of color in college discover internships and entry-level job opportunities from inclusive organizations. The platform allows users to search job listings by keyword and location and view detailed information about each opportunity in one place.

**Target Browsers:**
Chrome (desktop)
Firefox (desktop)
Safari (desktop)
